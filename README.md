# 🏃 AI Running Coach

**Персональный ИИ-тренер по бегу (AI-powered personal running coach)**

Система анализирует тренировки из TCX/FIT-файлов (формат не привязан к конкретным часам — подойдут Garmin, Coros, Polar и любые другие) и поддерживает **автоматическую синхронизацию с Coros Training Hub** — достаточно нажать одну кнопку, чтобы подтянуть новые беговые тренировки с часов Coros. Определяет тип бега (интервальная, темповая, восстановительная, длительная), отслеживает пульсовые зоны, набор высоты, погодные условия и динамику веса. Все операции логируются; данные тренировок можно безопасно удалять с возможностью восстановления.

## Концепция (Concept)

Накопить знания о тренировках спортсмена и с учётом статистики давать рекомендации на неделю, формировать план тренировок, хвалить за успехи и не давать перетренироваться — как настоящий тренер.

### Основные возможности (Features)

- **Автосинхронизация с Coros** — все беговые тренировки с Coros Training Hub одной кнопкой (FIT через неофициальный API); фоновая загрузка с прогресс-баром
- **Загрузка TCX/FIT** — парсинг файлов любых часов (дистанция, пульс, высота, GPS, каденс, калории; без привязки к бренду)
- **Автоматическая очистка ошибок GPS** — обнаружение и удаление GPS-скачков и нереального темпа с пересчётом дистанции; лог удалённых участков
- **Классификация тренировок** — автоматическое определение типа: интервальная, темповая, длительная, восстановительная
- **Пульсовые зоны Z1–Z5** — расчёт времени в каждой зоне относительно ЧССмакс
- **Погода** — данные с Open-Meteo (температура, WMO иконка) по GPS-координатам
- **Высота** — набор и спуск по каждому сегменту и за тренировку
- **График пульса/темпа** — интерактивный Chart.js на странице тренировки
- **Статистика** — сводка за неделю и месяц (км, время, пульс, типы тренировок)
- **Вес** — отслеживание динамики с графиком и таблицей
- **Восстановление (Coros)** — синхронизация ежедневных метрик здоровья: HRV, пульс покоя, уровень усталости, тренировочная нагрузка, готовность; отображается на главной странице и в детальном просмотре тренировки
- **Навигация по годам/месяцам** — фильтрация тренировок по периоду
- **Шифрование пароля** — пароль Coros хранится в зашифрованном виде (Fernet)
- **Логирование** — ротация логов, просмотр через браузер (`/logs`)
- **Безопасное удаление** — при удалении тренировки метаданные сохраняются; при повторной загрузке — модал подтверждения
- **Веб-интерфейс** — локальный сервер FastAPI с HTML/Chart.js; systemd‑сервис с автозапуском

## Стек технологий (Tech Stack)

- **Python 3** + **FastAPI** — сервер
- **SQLite** + **SQLAlchemy** — база данных
- **Chart.js** — графики на фронтенде
- **Open-Meteo API** — погода (бесплатно, без API-ключа)
- **timezonefinder** — определение часового пояса по GPS

## Быстрый старт (Quick Start)

```bash
# Установка зависимостей (Install dependencies)
cd running-coach
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Запуск сервера (Start server)
uvicorn main:app --host 0.0.0.0 --port 8000

# Открыть в браузере (Open in browser)
# http://localhost:8000
```

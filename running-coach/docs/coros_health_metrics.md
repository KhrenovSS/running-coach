## HRV (Heart Rate Variability)

### Что это
Вариабельность сердечного ритма (HRV) измеряет колебания времени между двумя последовательными ударами сердца. Это нормальное "нерегулярное" состояние сердечного ритма.

### Почему HRV используется для оценки
HRV отражает состояние вегетативной нервной системы (ВНС / ANS — Autonomic Nervous System). ВНС состоит из двух ветвей:
- **Симпатическая** (возбуждение, нервозность, усталость) → HRV снижается, ЧСС увеличивается
- **Парасимпатическая** (расслабление, спокойствие, отдых) → HRV увеличивается, ЧСС уменьшается

Если одна из ветвей чрезмерно активна, баланс нарушается — это ненормальная ситуация.

- **Краткосрочный баланс (суточная HRV)** — реакция организма на текущий стресс
- **Долгосрочный баланс (HRV в течение длительного времени)** — общее состояние здоровья: хронический стресс, усталость или болезнь

### Ночная HRV
HRV, измеренная во время сна, отслеживает баланс ВНС в состоянии покоя. Позволяет принимать решения о работе, жизни и физических упражнениях.

**Назначение:**
- Делать перерывы при повышенном стрессе, чтобы не увеличивать нагрузку
- В хорошей форме — выполнять более сложные тренировки

### Как использовать
Постоянное ношение часов во сне даёт более точные данные.

1. **Среднее значение HRV за ночь** — RMSSD за весь период сна. RMSSD наиболее стабилен, меньше подвержен влиянию дыхания, чувствителен к изменениям неврологического статуса. COROS использует RMSSD.

2. **Нормальный диапазон HRV** — рассчитывается за последние 30 дней. Для первой нормы нужно минимум 5 ночей. Обновляется и уточняется по мере сбора данных.

3. **Результаты ночной HRV:**
   - **Повышенная** — HRV выше нормы. Обычно расслабление, но обратить внимание на другие физиологические факторы.
   - **Норма** — HRV в норме. Организм в порядке.
   - **Пониженная** — HRV немного ниже нормы. Стресс, подумать об отдыхе.
   - **Низкая** — HRV значительно ниже нормы. Сильный стресс, рекомендуется отдых.

### Ночная HRV и Восстановление
Три типа восстановления организма:
1. **Восстановление ВНС (нервная система)** — определяется по HRV. Восстанавливается быстрее всего.
2. **Восстановление энергии** — восстанавливается вторым.
3. **Восстановление мышц** — восстанавливается дольше всех.

Сочетание физического восстановления и ночной HRV даёт точную оценку:
- **HRV в норме, восстановление низкое** → нервная система в порядке, но мышцы/энергия ещё не восстановились. Фокус на рекомендации по восстановлению.
- **HRV в норме + физическая сила достаточна** → все системы восстанавливаются должным образом.

### Почему HRV колеблется в течение ночи
Нормально. HRV изменяется под влиянием стадий сна и циркадного ритма. Колебания в пределах десятков мс — норма.

### Причины низкой HRV
Травмы, болезни, курение, алкоголь, напряжённая работа, плохое настроение, интенсивные физические нагрузки → ВНС переходит в состояние стресса/усталости → HRV снижается.

### Как повысить HRV
- Дыхательные упражнения, медитация — временно повышают HRV
- Длительные аэробные тренировки — повышают уровень HRV в целом (улучшают вегетативную регуляцию и восстановление)

---

## RHR (Resting Heart Rate) — Пульс покоя

Пульс в состоянии полного покоя (сон). Чем ниже RHR, тем лучше тренированность сердечно-сосудистой системы. Коррелирует с HRV: при низкой HRV пульс обычно повышен (симпатика доминирует).

---

## TiredRate / TiredRateNew — Уровень усталости

Оценка текущего уровня усталости организма. Чем выше значение, тем больше усталость. Зависит от тренировочной нагрузки, качества сна, стресса.

---

## Training Load — Тренировочная нагрузка

Кумулятивная нагрузка за день. Высокая → организм под стрессом, нужно восстановление.

---

## Performance / Readiness — Готовность к нагрузкам

Интегральный показатель восстановления:
- Сочетает восстановление ВНС (HRV), энергии и мышц
- Чем выше — тем готовность к тренировке больше
- Если HRV в норме, но готовность низкая → мышцы и энергия ещё восстанавливаются
- Если HRV в норме и готовность высокая → все системы в порядке

---

## ATI / CTI — Acute/Chronic Training Impulse

- **ATI** (Acute Training Impulse) — нагрузка за последние 7 дней (краткосрочная)
- **CTI** (Chronic Training Impulse) — нагрузка за последние 42 дня (долгосрочная)
- Соотношение ATI/CTI показывает, перетренирован ли спортсмен или недотренирован

---

## Stamina Level — Уровень выносливости

Оценка аэробной выносливости на основе тренировок и физиологических показателей.

# Численные пороги и правила для ИИ-анализа

## 1. Поля в БД (DailyMetrics)

| Поле | Тип | Ед.изм | Диапазон (наблюдаемый) | Описание |
|---|---|---|---|---|
| `avg_sleep_hrv` | Float | мс | 30–75+ | Средняя HRV за ночь (RMSSD) |
| `sleep_hrv_baseline` | Float | мс | 30–55 | 30-дневное скользящее среднее HRV |
| `sleep_hrv_sd` | Float | мс | NULL (пока не отдаётся) | Стандартное отклонение baseline |
| `rhr` | Integer | уд/мин | 40–65 | Пульс покоя (ночной) |
| `tired_rate` | Integer | у.е. | −10..+10 | Уровень усталости |
| `training_load` | Float | у.е. | 0–500+ | Дневная тренировочная нагрузка |
| `training_load_ratio` | Float | — | 0–2+ | Отношение нагрузки к норме |
| `performance` | Float | у.е. | −2..+2 | Интегральная готовность |
| `ati` | Integer | у.е. | 0–500+ | Acute Training Impulse (7 дней) |
| `cti` | Integer | у.е. | 0–2000+ | Chronic Training Impulse (42 дня) |

## 2. Классификация HRV

```
baseline = sleep_hrv_baseline  (30-дневное скользящее среднее)
sd = sleep_hrv_sd             (если NULL → sd = baseline * 0.2)

if hrv > baseline + sd        → "Повышенная"   (расслабление / возможна болезнь)
if hrv >= baseline - sd       → "Норма"         (всё в порядке)
if hrv >= baseline - 2*sd     → "Пониженная"    (стресс, подумать об отдыхе)
if hrv < baseline - 2*sd      → "Низкая"        (сильный стресс, обязательный отдых)
```

**Корректировка для разных уровней тренированности:**
- Новички: baseline = 20–40 мс, sd ≈ 4–8 мс
- Средний уровень: baseline = 40–65 мс, sd ≈ 8–15 мс
- Спортсмены: baseline = 60–100+ мс, sd ≈ 12–20+ мс

## 3. Классификация TiredRate (усталость)

```
if tired_rate <= −5  → "Низкая"      (организм восстановлен, готов к нагрузкам)
if tired_rate <= 0   → "Умеренная"   (нормальное состояние)
if tired_rate > 0    → "Высокая"      (нужен отдых)
```

## 4. Классификация Performance (готовность)

```
if performance > 0.5  → "Готов к тренировкам"     (полное восстановление)
if performance > −0.5 → "Умеренная готовность"    (можно тренироваться, но осторожно)
if performance ≤ −0.5 → "Требуется отдых"         (недовосстановление)
```

## 5. Классификация Training Load (нагрузка)

```
if training_load < 50   → "Лёгкая"
if training_load < 150  → "Средняя"
if training_load ≥ 150  → "Высокая"
```

## 6. RHR-аномалии

```
# Персональная базовая норма RHR (30-дневное скользящее среднее)
rhr_baseline = rolling_30d_avg(rhr)

if rhr > rhr_baseline + 5:
    → "Пульс покоя повышен на {rhr - rhr_baseline} уд/мин. Признак недовосстановления."
if rhr > rhr_baseline + 10:
    → "Критическое повышение пульса покоя. Возможна болезнь или перетренированность."
if rhr < rhr_baseline - 3:
    → "Пульс покоя аномально низкий. Проверить самочувствие."
```

## 7. ATI/CTI ratio (перетренированность)

```
ratio = ati / cti   (если cti > 0)

if ratio > 1.3   → риск перетренированности (нагрузка растёт быстрее адаптации)
if ratio > 1.5   → высокая вероятность перетренированности, нужна разгрузочная неделя
if ratio < 0.8   → недостаточная нагрузка (тренировки слишком лёгкие)
if 0.8-1.3       → оптимальный диапазон
```

## 8. Комбинированные правила для рекомендаций

```
# Правило 1: Нервная система под стрессом
IF hrv_status IN ("low", "very_low")
    → "Нервная система под стрессом [{hrv_status}]. 
       Рекомендуется отдых или лёгкая аэробная сессия (Z1–Z2). 
       Исключить интервальные и силовые тренировки."

# Правило 2: Усталость при нормальной HRV
IF tired_label == "high" AND hrv_status == "normal"
    → "Нервная система в порядке, но мышцы/энергия ещё не восстановились. 
       Провести лёгкую восстановительную тренировку или активный отдых."

# Правило 3: Полное восстановление
IF performance > 0.5 AND hrv_status == "normal"
    → "Полное восстановление. Можно проводить интенсивную тренировку 
       (интервалы, темповая, длительная)."

# Правило 4: Недовосстановление по пульсу
IF rhr_anomaly == "elevated"
    → "Пульс покоя повышен на {delta} уд/мин. 
       Организм не восстановился после предыдущей нагрузки.
       Снизить интенсивность на 20–30%."

# Правило 5: Риск перетренированности
IF ati_cti_ratio > 1.3
    → "Соотношение ATI/CTI = {ratio:.1f} — нагрузка растёт быстрее адаптации.
       Рекомендуется разгрузочная неделя (снижение объёма на 40–50%)."

# Правило 6: Недостаточная нагрузка
IF ati_cti_ratio < 0.8 AND training_load < 50 FOR last_7_days
    → "Недостаточная тренировочная нагрузка. Организм теряет адаптацию 
       (detraining). Увеличить недельный объём."

# Правило 7: Положительная динамика
IF hrv_upward_trend(last_30_days) AND rhr_downward_trend(last_30_days)
    → "Положительная динамика: HRV растёт, пульс покоя снижается. 
       Аэробная база улучшается. Продолжать текущий план."

# Правило 8: Сравнение с предыдущим днём
IF hrv_status == "normal" AND yesterday_hrv_status IN ("low", "very_low")
    → "HRV восстановилась до нормы. Организм справился со стрессом."

IF hrv_status IN ("low", "very_low") AND yesterday_hrv_status == "normal"
    → "HRV упала ниже нормы. Возможные причины: 
       перетренированность, недосып, алкоголь, болезнь."
```

## 9. Метрики для отслеживания трендов (30+ дней)

| Тренд | Что означает | Рекомендация |
|---|---|---|
| HRV ↑, RHR ↓ | Улучшение аэробной базы | Продолжать план, можно повышать нагрузку |
| HRV ↓, RHR ↑ | Хронический стресс / перетрен | Снизить объём на 30%, добавить отдых |
| HRV стабильна, RHR стабилен | Плато / адаптация | Пора менять программу (прогрессия) |
| HRV ↓, RHR ↓ | Возможна болезнь / переутомление | Полный отдых 2–3 дня |
| TiredRate ↑ 3+ дня подряд | Кумулятивная усталость | Разгрузочная неделя |
| Performance < −0.5 3+ дня | Хроническое недовосстановление | Пересмотреть режим |

## 10. Расчёт скользящих метрик в коде

```python
# Пример: скользящее среднее HRV за N дней (N=7, 30, 42)
def rolling_avg(records, field, days):
    """records: список DailyMetrics, сортированный по date DESC"""
    recent = [r for r in records if r.date >= cutoff]  # cutoff = today - days
    values = [getattr(r, field) for r in recent if getattr(r, field) is not None]
    return statistics.mean(values) if values else None

# Пример: тренд (линейная регрессия по датам)
def trend(records, field, days=30):
    """Возвращает наклон (slope): >0 рост, <0 падение"""
    recent = [r for r in records if r.date >= cutoff]
    points = [(r.date.toordinal(), getattr(r, field)) for r in recent
              if getattr(r, field) is not None]
    if len(points) < 5:
        return None
    xs, ys = zip(*points)
    slope, _ = np.polyfit(xs, ys, 1)  # или ручная реализация OLS
    return slope  # ед./день, например мс/день для HRV

# Пример: обнаружение аномалий
def anomaly(rhr, rhr_baseline):
    delta = rhr - rhr_baseline
    if delta > 10:
        return 'critical'
    elif delta > 5:
        return 'elevated'
    elif delta < -3:
        return 'low'
    return 'normal'
```

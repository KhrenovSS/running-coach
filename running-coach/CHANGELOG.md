# История проекта Running Coach

## 21.06.2026 (вечер)
- **График пульса/темпа — исправление ступенек**: `dist_km` повышен с 1 до 3 знаков (0.100 → 0.101 км) — каждая точка на уникальной X-позиции, устранены столбики. Добавлен `cubicInterpolationMode: 'monotone'` (монотонный кубический сплайн = полиномиальная интерполяция). Tension снижен с 0.8 до 0.4. Даунсэмплинг до ~200 точек. Ось X — числовая линейная шкала, шаг сетки 0.25 км. Пульс — треугольное взвешивание (40с / 5с), float 1 знак. Темп — окно 250м + двойной проход треугольного фильтра 45с, 2 знака, нуль осцилляций.
- **Динамика веса**: Добавлена модель `WeightMeasurement` (id, weight_kg, measured_at). При сохранении веса в настройках запись добавляется в историю. Дубликаты при том же весе не создаются.
- **График веса (Chart.js spline)**: Скрыт, открывается по клику на вес в панели настроек. Показывает таблицу всех взвешиваний и плавный график (tension=0.4).
- **Вес по умолчанию**: Изменён с 70 на 85 кг.
- **Миграция БД**: При старте сервера создаётся таблица `weight_measurements` и переносится текущий вес как первая точка.

## 21.06.2026
- **Высота (elevation)**: Парсинг `AltitudeMeters` из TCX. В каждом сегменте показывается набор/спуск (↑м/↓м). Суммарный набор/спуск — в карточке тренировки.
- **Часовой пояс**: Определяется через библиотеку `timezonefinder` по GPS-координатам. Время конвертируется из UTC в локальное (например, Москва → +03:00). В БД сохраняется наивное локальное время.
- **Погода**: Интегрирован Open-Meteo Archive API (бесплатно, без API-ключа). По координатам и дате подтягивается почасовая температура. Средняя температура за тренировку — на главном экране. Температура по сегментам — в детальном просмотре. Кэширование в памяти.
- **Формат времени**: Длительность тренировки отображается в часах и минутах (например, `1ч 15мин` или `42мин`).
- **Статистика на главной**: Добавлены карточки «Неделя (7 дней)» и «Месяц (30 дней)» с отступом от последней тренировки. В каждой: общий километраж, общее время, столбчатая диаграмма времени по пульсовым зонам (Z1-Z5), количество тренировок по типам.
- **Компактная загрузка**: Кнопка «Загрузить TCX» вынесена в панель настроек (рядом с кнопкой «Настройки»), убран громоздкий блок загрузки.
- **Диапазоны пульса**: В столбцах Z1-Z5 добавлен диапазон ударов в минуту (например, Z2 (125–142)).

## База
- FastAPI + SQLite
- Парсинг TCX с Garmin
- Классификация: интервальная / темповая / длинная / восстановительная
- Веб-интерфейс: список тренировок, детальный просмотр, загрузка TCX, настройки (ЧССмакс, вес)
- Пульсовые зоны: Z1 (≤70%), Z2 (70-80%), Z3 (80-87%), Z4 (87-93%), Z5 (>93%)
